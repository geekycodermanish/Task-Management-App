Task Manager App

Task Manager is a full-stack application built to efficiently manage tasks. It utilizes React.js for the frontend and Node.js with Express for the backend. Key features include:

    Task List UI: Intuitive interface for managing tasks.
    Excel File Upload: Easily upload tasks via Excel files.
    REST API Integration: Seamless communication between frontend and backend.
    CRUD Operations: Perform Create, Read, Update, and Delete operations on tasks.
    PDF Attachment: Attach PDF files to tasks for reference.

Getting Started

    Clone the repository.
    Install dependencies for both frontend and backend (npm install).
    Start the frontend (npm start) and backend (npm run server) servers.
    Open your browser and navigate to the provided URL.

Technologies Used

    React.js
    Node.js
    Express
    MongoDB (optional)
    ExcelJS
    PDFKit
